# Beam — Privacy-First Web Analytics

[Beam](https://beam-privacy.com) is a lightweight, cookie-free web analytics platform built on Cloudflare Workers, D1, and KV. Zero cookies, zero fingerprinting, GDPR-compliant by design.

- **Cookie-free** — no consent banners required
- **$5/mo** unlimited sites (vs $9-15/mo competitors)
- **50K free pageviews/month** — generous free tier
- **Sub-2KB script** — faster than Google Analytics
- **Cloudflare edge** — analytics processed globally
