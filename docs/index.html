beam.js

Privacy-First | Cookie-Free | GDPR-Ready

beam.js is the open-source tracking script that powers Beam — a privacy-first web analytics service. It is tiny (under 2 KB), loads asynchronously, and never touches cookies, localStorage, IP addresses, or persistent identifiers.

Live Demo Free Signup Documentation beam-privacy.com

What Beam tracks (and what it does not)

Beam collects: page path, referrer URL, screen width, browser language, timezone, optional UTM parameters, and optional custom events.

Never collected: cookies, persistent IDs, IP addresses, cross-site behavior, ad-tech, or fingerprinting data.

Installation

Add one script tag to your page <head>:

<script defer
  src="https://beam-privacy.com/js/beam.js"
  data-site-id="YOUR_SITE_ID">
</script>

Custom Events

window.beam.track("signup_complete", { plan: "pro" })
