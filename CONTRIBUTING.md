# Contributing

Thanks for taking a look at `beam.js`.

This repository covers the browser tracking script only. Beam's ingestion API, database, authentication, billing, and dashboards are part of the hosted Beam SaaS and are not published here.

## Scope

- Script behavior and payload shape
- Documentation improvements
- Privacy and security review feedback

## Not in scope

- Hosted backend changes
- Dashboard feature requests
- Billing or account-management changes

If you want to report an issue with the hosted product, contact `ralph@keylightdigital.dev`.
